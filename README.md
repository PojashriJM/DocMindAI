# DocMindAI

An AI-powered document intelligence platform that allows users to upload PDFs, notes, and files, then ask questions, summarize content, generate flashcards, and extract insights instantly.

## Features
- Chat with PDF/Documents
- AI Summarization
- Flashcards Generator
- Quiz Generation
- Semantic Search
- Clean UI
- Dark / Light Mode

## Tech Stack
- Next.js
- React.js
- Tailwind CSS
- OpenAI API / Gemini API
- Vector DB
- PDF Parser

## Future Scope
- Voice Interaction
- Multi-language Support
- Team Collaboration
